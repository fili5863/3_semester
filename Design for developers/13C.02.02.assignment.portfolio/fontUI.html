13C.01.01 Fonts and UI components

In this assignment, I had to create a font hierarchy, with one or two fonts.

We started by verifying that the hierarchy was working using a font-tester, provided by my teacher. Then I applied it to the website.
I also learned about the importance of font-sizing, line-height, margin, and kerning.

The second part of this assignment, I had to improve the UI for the website, which I achieved with basic CSS.

I used flex and grid to make the cards more user friendly and made the buttons an orange color with hover effects, so the user knows that the element is interactive.

Lastly, I gave the website some basic colors, to make it look less bland.
