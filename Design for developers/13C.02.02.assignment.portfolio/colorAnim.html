13C.01.02 Colors and animations

In this assignment, I learned how to apply a color palette to a basic website.

I used Huemint (https://huemint.com/) to get inspiration and like the last assignment, I was provided with a color tester file from my teacher to check if the color palette I had chosen, was applicable in other settings. I also used Adobe’s Contrast Analyzer to see if the color contrast worked in different sizes, such as paragraphs and smaller graphic elements. After using both methods, I applied the color palette to the website I was given using CSS variables for the colors.

The second part of this assignment, I had to create a “Dark theme”, I used a combination of JavaScript and CSS to achieve this. I made the button itself add a “dark mode” class to all the elements with the applied color palette and mixed the colors in another way to make it easier on the eyes.

In the third and last part of this assignment, I added some basic animations to the site, such as a progress bar in the top, so the user knows how far they are down on the page and a fade in animation when the page is loaded. Once again with some basic JavaScript and CSS.
